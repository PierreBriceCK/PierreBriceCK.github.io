Description du projet


Dans ce projet, une application web de suivi des absences a été développée avec Symfony pour permettre aux utilisateurs de s’authentifier et de gérer les absences au sein de l’interface dédiée du Lycée 44.




Dans le cadre du projet Lycee 44, quatre outils principaux ont été développés pour faciliter la gestion des absences :


1. Etudiant

Cette interface permet de gérer les informations des étudiants, ainsi que leur rattachement à une classe.​





2. Classe

Cette interface permet de gérer les détails des classes, ainsi que leur association avec les étudiants.​





3. Matiére

Cette interface permet de gérer les informations des matières, telles que leur nom, leur coefficient et leur volume horaire.​​





4. Absence

Cette interface permet d’enregistrer et de gérer les absences des étudiants, avec la possibilité d’indiquer si elles sont justifiées.​





Autre

Dans ce projet, un système de déconnexion a été mis en place avec Symfony pour permettre aux utilisateurs de se déconnecter de l’interface de suivi des absences. Cette fonctionnalité renforce la sécurité en mettant fin à la session active, évitant ainsi tout accès non autorisé aux données sensibles.
